# Schopenhauer.nvim

Light Neovim colorscheme.

![preview](preview.png)

## Installation

### [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
  "dzania/schopenhauer.nvim",
  priority = 1000,
  config = function()
    vim.cmd("colorscheme schopenhauer")
  end,
}
```
